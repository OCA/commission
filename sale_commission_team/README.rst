Configuration
=============

The only configuration required is having the proper agents set on Sale Order Teams



Usage
=====

The 2 features currently in this model are setting Sale Order Line Agents by default when a Sale Order Line is created.

The other feature is upon swithing the Sale Order Team it will autopopulate the Sale Order Lines with new Agents from the new Team

Known issues / Roadmap
======================

* Cannot have the same agent in 2 separate teams
